## 자바스크립트의 데이터 타입
자바스크립트의 데이터 타입은 기본형과 참조형으로 나눌 수 있습니다.  
기본형은 값이 담긴 주솟값을 바로 복제하는 반면 참조형은 값이 담긴 주솟값들로 이루어진 묶음을 가리키는 주솟값을 복제합니다.


<br/>

## 변수(variable)와 식별자(identifier)
변수는 변경가능한 데이터가 담길 수 있는 공간이고, 식별자는 어떤 데이터를 식별하는 데 사용하는 이름(변수명)입니다.

<br/>

### 변수 선언
```
let a;
```
메모리에서 비어 있는 공간 하나를 확보하고, 공간의 이름을 a라고 지정합니다.   
사용자가 a에 접근하고자 하면 메모리에서 a라는 이름을 가진 주소에 검색해 해당 공간에 담긴 데이터를 반환합니다.

### 데이터 할당
```
let a;  변수 선언
a = 1;  데이터 할당
```

1. 변수 영역에서 빈 공간을 확보하고, 식별자 a를 지정  
2. 데이터 영역의 빈 공간에 데이터 1을 저장  
3. 변수 영역에서 식별자 a를 검색  
4. 1을 저장한 데이터 영역의 주소를 식별자 a를 가진 공간에 저장  
> 변수 영역에 값을 직접 대입하지 않는 이유가 무엇일까?  
데이터 변환을 자유롭게 할 수 있게 하고, 메모리를 효율적으로 관리하기 위해서입니다.   
예를 들어, 미리 확보한 공간 내에서만 데이터 변환을 할 수 있다면 변환한 데이터를 다시 저장 하기 위해서 먼저 확하된 공간을 변환된 데이터 크기에 맞게 늘리는 작업을 해야합니다.

## 기본형 데이터와 참조형 데이터

### 불변값
기본형 데이터는 모두 불변값입니다.
불변값을 새로 만드는 동작을 통해서만 변경됩니다. 그리고 한 번 만들어진 값은 영원히 변하지 않습니다.

### 가변값

참조형 데이터의 상당수가 가변값이지만 변경 불가능하게 만들거나 불변값으로 활용할 수 있습니다. 참조형 데이터는 기본형 데이터와는 다르게 객체의 변수 영역이 별도로 존재합니다.데이터 영역은 기존의 메모리 공간을 그대로 활용하는데, 저장된 값들은 모두 불변값입니다.하지만 변수 영역은 다른 값을 대입할 수 있습니다.이러한 점으로 인해 참조형 데이터는 불변하지 않습니다. (가변값)
여기서 가변값이란 참조형 데이터 자체를 변경하는 경우가 아니라 그 내부의 프로퍼티를 변경할 때만 성립합니다.

### 변수 복사 비교
기본형은 값을 복사하고 참조형을 주솟값을 복사한다고 알려져 있지만 사실을 어떤 데이터 타입이든 변수에 할당하기 위해서는 주솟값을 복사해야 합니다. 그렇기 때문에 자바스크립트의 모든 데이터 타입은 참조형 데이터입니다. 하지만 기본형은 주솟값을 복사하는 과정이 한번이고, 참조형은 한 단계를 더 거치게 됩니다.
